# تعبئة اسم الصنف من الكود

هذه المرحلة تضيف:
- عند مسح كود صنف، يبحث التطبيق في الأصناف المحفوظة سابقًا.
- إذا وجد نفس الباركود يملأ اسم الصنف والتصنيف والوحدة والأسعار.
- إذا كان الكود جديدًا، يطلب إدخال الاسم مرة واحدة.
- عند حفظ الصنف، يتعلم التطبيق الربط بين الكود واسم الصنف على الجهاز الحالي.
- ملاحظة: الباركود نفسه لا يحتوي اسم المنتج غالبًا؛ لذلك لا يمكن استخراج الاسم لأول مرة إلا من قاعدة بيانات أو OCR لاحقًا.
