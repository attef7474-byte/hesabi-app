# Hesabi App 1.0.91 - Items Mobile Table / Search Fix

## Scope
واجهة صفحة الأصناف والمخزون فقط.

## Changes
- جعل أيقونات صفحة الأصناف العلوية بنفس نمط أيقونات صفحة الإعدادات.
- تثبيت التبويب الافتراضي على الأصناف.
- عرض أول 10 أصناف افتراضيًا في جدول.
- إضافة أزرار السابق/التالي أعلى الجدول.
- جعل جدول الأصناف داخل تمرير أفقي محدود بدل تمدد الصفحة كلها.
- منع إعادة رسم الصفحة مع كل حرف في البحث حتى لا تختفي لوحة المفاتيح.
- إضافة زر بحث يدوي، ويمكن الضغط Enter أيضًا.

## Not changed
- لم يتم تغيير Firestore.
- لم يتم تغيير إضافة/تعديل/حذف الأصناف.
- لم يتم تغيير الاستيراد والتصدير.
- لم يتم تغيير السداد أو الفواتير أو الطلبات.

## Checks
- node --check للملفات المعدلة والجديدة.
- فحص JSON للـ manifest و android-update.
- فحص runtime مجمع حسب ترتيب manifest.

## Device testing
لم يتم اختبار APK على هاتف فعلي داخل هذه البيئة.
