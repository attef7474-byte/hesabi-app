# 1.0.38 - Keyboard Stability Fix

تم إصلاح مشكلة اختفاء لوحة المفاتيح بعد كل حرف في:

- بحث نموذج طلب الشراء للعميل.
- كتابة الرسائل بين العميل والتاجر.

الإصلاح يمنع إعادة بناء الصفحة كاملة أثناء الكتابة، ويحدث نتائج الشراء فقط بدون فقدان تركيز الحقل. كما يمنع تحديثات Firestore من إعادة رسم الصفحة أثناء الكتابة في الحقول الحساسة.
