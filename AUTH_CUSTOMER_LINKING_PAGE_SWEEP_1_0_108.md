# Hesabi App 1.0.108 - Login + Customer Trader Linking Page Sweep

## النطاق
- التسجيل والدخول.
- اختيار نوع الحساب تاجر/عميل.
- تهيئة التاجر.
- تهيئة العميل وربطه بالتاجر.
- صفحة متاجري للعميل.

## ما تم تثبيته
- تبويبات الدخول SMS / الدخول القديم.
- حقول رقم الهاتف وكود التحقق.
- اختيار تاجر / عميل.
- أزرار إنشاء/استرجاع محل التاجر.
- إضافة كود تاجر للعميل.
- البحث عن متجر وربط المتاجر المحددة.
- صفحة متاجري: عرض المتاجر المرتبطة، تبديل المتجر، إضافة تاجر، استرجاع ربط سابق.
- حماية العرض على الهاتف من التمدد الأفقي.
- إضافة فحص ذاتي للمرحلة وربطه مع فحص runtime الكامل.

## ما لم يتم تغييره
- لم يتم تغيير منطق Firebase Authentication.
- لم يتم تغيير منطق Firestore.
- لم يتم تغيير طريقة إنشاء المتجر أو ربط العميل فعليًا.
- لم يتم تغيير الطلبات أو الفواتير أو السداد.
