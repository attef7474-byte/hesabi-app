# استيراد وتصدير الأصناف من Excel

هذه المرحلة تضيف:
- ملف hesabi-items-1000-template.xlsx يحتوي 1000 صنف.
- زر استيراد Excel داخل صفحة الأصناف.
- زر تصدير Excel للأصناف الحالية.
- رابط تحميل النموذج من داخل التطبيق.
- عند الاستيراد يتم تحديث الصنف إذا وجد نفس barcode/imei/serial، أو إضافته إذا كان جديدًا.
- يتم حفظ ربط الكود باسم الصنف للتعرف عليه لاحقًا عند المسح.
- الإصدار: 1.0.12.
