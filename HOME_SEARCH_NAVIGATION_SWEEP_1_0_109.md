# Hesabi App 1.0.109 - الرئيسية + البحث العام + شريط التنقل

## النطاق
- تثبيت صفحة الرئيسية.
- تثبيت البحث العام.
- تثبيت شريط التنقل السفلي.
- حماية العرض على الهاتف.

## ما تم
- استبدال عرض الرئيسية بعرض مختصر: عنوان، مؤشرات سريعة، وأيقونات الصفحات.
- تثبيت أزرار الرئيسية وربطها بالصفحات بدون تضارب مع تبويبات الصفحات الداخلية.
- تعديل البحث العام ليعمل بزر بحث أو Enter بدل إعادة رسم الصفحة مع كل حرف.
- زر مسح البحث يعيد النوع إلى كل النتائج.
- زر فتح المكان في نتائج البحث ينقل للصفحة الصحيحة.
- زر كشف الحساب يظهر للنتائج المرتبطة بعميل.
- شريط التنقل السفلي أصبح يربط أزراره داخل `#navInner` فقط حتى لا يكسر تبويبات الصفحات الأخرى.
- إضافة زر البحث في شريط التنقل.
- حماية التمدد الأفقي في الرئيسية والبحث والشريط السفلي.

## لم يتم تغييره
- لم يتم تغيير Firestore.
- لم يتم تغيير الطلبات أو الفواتير أو السداد أو المخزون.
- لم يتم تغيير منطق تسجيل الدخول أو ربط العميل بالتاجر.
