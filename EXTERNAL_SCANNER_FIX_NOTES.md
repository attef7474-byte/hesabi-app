# تحسين ماسح كود الصنف

- إصلاح خطأ Java في jsQuote.
- فتح تطبيق الماسح الخارجي كخيار أساسي.
- عند قراءة رقم IMEI من ملصق يحتوي عدة باركودات، لا يضعه تلقائيًا ككود صنف، بل يحذّر المستخدم ويضعه في الملاحظات.
- اسم الصنف لا يمكن معرفته من باركود CODE_128 إلا إذا كان هناك قاعدة بيانات تربط الكود بالموديل.
