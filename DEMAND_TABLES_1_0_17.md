# Hesabi 1.0.17 - Demand Tables Update

تم تطبيق نظام الجداول حسب الطلب للصفحات التي تحتوي بيانات كثيرة.

## السلوك الجديد
- عند فتح الصفحة لا تعرض البيانات تلقائياً.
- يظهر شريط بحث وزر عرض البيانات.
- يتم عرض أول 50 سجل فقط.
- يوجد زر تحميل المزيد.
- البحث يعرض النتائج حسب الطلب.

## الصفحات المحدثة
- الأصناف
- العملاء
- أصناف البيع للعميل
- الطلبات
- الفواتير
- السداد
- كشف الحساب
- سجل العمليات
- المخزون
- حركات المخزون
- المرتجعات
- التحصيلات
- الجداول/الأقساط

## ملاحظة
هذا التحديث يخفف الواجهة ويمنع ازدحام الشاشة عند وجود 1000 صنف أو أكثر.
