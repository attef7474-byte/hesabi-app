# إصلاح استيراد Excel

سبب المشكلة:
- Android WebView file chooser كان يسمح عمليًا بالصور والصوت فقط، لذلك ملف Excel لا يظهر أو لا يتعرف عليه التطبيق.
- تحميل مكتبة Excel كان يعتمد على مصدر CDN واحد.

الإصلاح:
- دعم صيغ xlsx/xls/csv في file chooser داخل MainActivity.
- تحسين تحميل SheetJS من عدة مصادر بديلة.
- رسائل أوضح عند اختيار ملف غير Excel.
- الإصدار: 1.0.13.
