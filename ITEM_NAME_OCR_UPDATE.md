# OCR لاسم الصنف

هذه المرحلة تضيف:
- زر قراءة اسم الصنف OCR داخل نموذج الصنف.
- فتح كاميرا الهاتف لتصوير ملصق الصنف.
- استخدام ML Kit Text Recognition داخل APK.
- استخراج الموديل مثل HUAWEI B535-932a إذا كان النص واضحًا.
- إضافة النص المقروء إلى الملاحظات كمرجع.
- رفع إصدار APK إلى 1.0.11.

مهم:
OCR يعمل داخل APK بعد البناء والتثبيت، ولا يعمل من المتصفح العادي.
