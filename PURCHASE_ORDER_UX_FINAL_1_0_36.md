# Hesabi 1.0.36 - Purchase Order UX Final Fix

- تحويل صفحة شراء العميل إلى مسار واضح: بحث، تصنيف، إضافة للطلب، سلة، نموذج طلب شراء.
- إضافة زر واضح لكل صنف: إضافة للطلب.
- إضافة نموذج طلب شراء ظاهر أسفل الصفحة مع تعديل الكميات ونوع الدفع والملاحظة وإرسال الطلب.
- إضافة زر سلة ثابت للانتقال مباشرة إلى نموذج الطلب.
- دعم الصفحات السابق/التالي للأصناف بدلاً من جدول طويل مربك.
- احترام سياسات التاجر: إخفاء السعر، إخفاء الكمية، منع طلب غير المتوفر.
