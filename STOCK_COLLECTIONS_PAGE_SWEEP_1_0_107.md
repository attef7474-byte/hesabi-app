# Hesabi App 1.0.107 - Stock + Daily Collections Page Sweep

## Scope
- صفحة المخزون.
- صفحة التحصيل اليومي.

## Changes
- تثبيت تبويبات المخزون: ملخص / المخزون / منخفض / الحركات.
- تثبيت البحث داخل المخزون بدون إعادة رسم أثناء الكتابة.
- إضافة أزرار صريحة للبحث والمسح والسابق والتالي.
- إضافة أزرار تسوية المخزون وتعديل الصنف من جدول المخزون.
- إضافة تصدير CSV للأصناف وحركات المخزون.
- تثبيت تبويبات التحصيل اليومي: اليوم / معلق / مقبول / مرفوض.
- تثبيت فلتر التاريخ وطريقة الدفع.
- تثبيت البحث داخل التحصيل بدون إغلاق لوحة المفاتيح.
- تثبيت فتح الإيصال وقبول/رفض السداد من صفحة التحصيل.
- إضافة مشاركة وتصدير CSV للتحصيل.
- حماية جداول المخزون والتحصيل من التمدد الأفقي على الهاتف.
- ربط الفحص الذاتي داخل فحص runtime الكامل.

## Safety
- لا تغيير في منطق Firestore.
- لا تغيير في منطق السداد أو الفواتير أو الطلبات.
- لا تغيير في معادلات المخزون؛ يتم استدعاء دوال التسوية/القبول/الرفض الموجودة كما هي.
