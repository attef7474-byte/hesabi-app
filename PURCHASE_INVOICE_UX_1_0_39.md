# Hesabi App 1.0.39 - Purchase Invoice UX Final

- Removed customer instructional text from the purchase page.
- Purchase page now opens directly on the purchase order invoice form.
- Invoice table always includes one empty row for item entry.
- Customer can type an item name and choose matching store items from suggestions.
- Quantity defaults to 1 and can be changed from the invoice row.
- Added a separate Add Items page with search, category filter, and a default list of 10 popular items.
- Selecting an item from Add Items adds it directly to the invoice/cart.
- Save returns to the invoice for review and sending.
