# تنظيم صفحة الأصناف

تمت إضافة شريط أيقونات أعلى صفحة الأصناف:
- الأصناف
- إضافة صنف
- تعديل صنف
- استيراد وتصدير

كل أيقونة تعرض جزءًا خاصًا بوظيفتها داخل صفحة الأصناف:
- قائمة الأصناف والمخزون.
- نموذج إضافة صنف مع الماسح و OCR.
- نموذج تعديل صنف محدد.
- أدوات Excel والحذف والتصدير.

الإصدار: 1.0.16.
