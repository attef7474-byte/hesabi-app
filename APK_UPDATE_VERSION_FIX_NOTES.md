# إصلاح تحديث APK وإظهار رقم الإصدار

- فصل زر تحديث الواجهات عن زر تحديث APK.
- إضافة فحص APK من android-update.json.
- إظهار إصدار الواجهات وإصدار APK المثبت داخل الإعدادات.
- زر تحديث APK يستدعي HesabiAndroid.openLatestApk داخل APK، أو يفتح رابط التنزيل عند استخدام المتصفح.
- رفع versionCode إلى 8 و versionName إلى 1.0.8.
