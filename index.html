إعداد Firebase Firestore

أدخل إعدادات Firebase Web App مرة واحدة. بعدها سيعمل التزامن بين هاتف العميل وهاتف التاجر عبر Firestore.

قواعد Firestore للتجربة فقط
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

هذه القواعد مفتوحة للتجربة فقط. لا تستخدمها لبيانات حقيقية بدون Firebase Authentication وقواعد أمان.

تسجيل الدخول الآمن

للنشر التجاري يجب الدخول بحساب آمن. استخدم رقم الهاتف وكلمة مرور. يجب تفعيل Email/Password من Firebase Authentication. إذا كانت شاشة الدخول لا تظهر عند فتح الملف المحلي، استخدم هذه النسخة المعدلة أو ارفع التطبيق على Firebase Hosting.

بعد الدخول سيحفظ التطبيق معرف المستخدم UID ويربط به المحل أو حساب العميل، وهذا هو أساس قواعد الأمان.

تهيئة أول تشغيل

اختر نوع المستخدم. العميل يرى مشترياته وسداداته فقط، والتاجر يرى إدارة المحل والطلبات.
