# Hesabi App 1.0.33 - إصلاح صلاحيات ربط العميل والبحث عن المتاجر

هذا التحديث يعالج ظهور رسالة Missing or insufficient permissions عند تهيئة العميل أو البحث عن متجر.

## الإصلاحات
- السماح للمستخدم المسجل بقراءة بيانات المتاجر الأساسية للبحث والاختيار.
- السماح للعميل الذي دخل برقم الهاتف الصحيح بقراءة/ربط سجل العميل الخاص بنفس رقم الهاتف.
- تحسين رسائل الخطأ داخل التطبيق وعدم عرض النص الإنجليزي الخام للمستخدم.
- رفع الإصدار إلى 1.0.33 / versionCode 33.

## مهم
يجب نشر firestore.rules مع هذا التحديث. إذا لم تُنشر القواعد ستبقى بعض الأجهزة تظهر خطأ صلاحيات.
