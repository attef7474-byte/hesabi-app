# حسابي التجاري 1.0.25 - صلاحيات التاجر والعميل

تم تعديل النسخة فوق 1.0.24 بدون رفع نهائي.

## الهدف
- العميل لا يستطيع إضافة أو تعديل أو حذف الأصناف.
- العميل لا يستطيع تعديل الأسعار أو المخزون أو السياسات.
- التاجر فقط يدير متجره وأصنافه وأسعاره ومخزونه وعملاءه.
- العميل يستخدم التطبيق للطلب والسداد والمراسلة ومتابعة كشف حسابه فقط.

## ما تم
- إضافة حارس صلاحيات داخل JavaScript للعمليات الحساسة.
- تحويل صفحة الأصناف عند العميل إلى عرض قراءة فقط.
- حماية حذف صنف واحد أو عدة أصناف من حساب العميل.
- حماية تعديل السعر السريع وإظهار/إخفاء الصنف من حساب العميل.
- حماية الاستيراد والتصدير والحذف الجماعي للأصناف.
- حماية صفحة السياسات والمخزون والعملاء والتقارير الإدارية.
- إضافة جدول توضيحي للصلاحيات داخل الإعدادات.
- تحديث ملف firestore.rules بقواعد تفصل صلاحيات التاجر والعميل.

> مهم: عند الرفع النهائي يجب نشر قواعد Firestore أيضًا حتى تكون الحماية من قاعدة البيانات وليس من الواجهة فقط.
