# حذف كل الأصناف السابقة

هذه المرحلة تضيف زرًا داخل صفحة الأصناف:
- حذف كل الأصناف السابقة

الحماية:
- يعمل للتاجر فقط.
- يطلب كتابة: حذف الأصناف
- ثم يطلب كتابة كود المحل للتأكيد.
- يحذف مجموعة items.
- يحذف stockLedger المرتبط بالمحل.
- يمسح ربط الأكواد المحلي itemCodeNames.
- بعدها يمكن استيراد ملف Excel الجديد.

الإصدار: 1.0.14
