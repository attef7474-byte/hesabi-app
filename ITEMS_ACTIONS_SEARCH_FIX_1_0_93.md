# ITEMS ACTIONS SEARCH FIX 1.0.93

## الهدف
إصلاح مشاكل صفحة الأصناف على الهاتف بعد تحويلها إلى جدول:
- التنقل بين أيقونات صفحة الأصناف لا يعمل.
- البحث لا يعرض الأصناف المطابقة.
- أزرار الإجراءات داخل جدول الأصناف مثل تعديل وسعر لا تعمل.

## ما تم تغييره
- إضافة `js/modules/32_items_actions_search_fix.js`.
- ربط تبويبات الأصناف بطريقة delegated event binding حتى تعمل بعد أي إعادة رسم للصفحة.
- تحسين بحث الأصناف ليطابق الاسم والباركود والكود والتصنيف والوحدة والأسعار والمخزون، مع تطبيع عربي بسيط.
- تثبيت أزرار الجدول:
  - تعديل.
  - سعر.
  - حفظ السعر.
  - إظهار/إخفاء.
  - حذف.
- إبقاء البحث بدون إعادة رسم أثناء الكتابة حتى لا تختفي لوحة المفاتيح.
- البحث يتم عند الضغط على زر بحث أو Enter.
- رفع الإصدار إلى 1.0.93 / build 93.

## ما لم يتم تغييره
- لم يتم تعديل Firestore.
- لم يتم تعديل منطق إضافة أو حفظ الصنف.
- لم يتم تعديل السداد أو الفواتير أو الطلبات.
- لم يتم تعديل قواعد الصلاحيات.

## الفحوصات المحلية
- node --check js/index_script.js
- node --check js/modules/32_items_actions_search_fix.js
