# MESSAGES_NOTIFICATIONS_MOBILE_FIX_1_0_90

## الهدف
إصلاح شكل صفحتي الرسائل والإشعارات على الهاتف بعد ظهور تمدد أفقي وجداول عريضة وانزياح في المحتوى كما ظهر في صور الاختبار.

## ما تم تغييره
- تحديث `js/modules/12_messages_helpers.js`.
- تحديث `js/modules/18_notifications_helpers.js`.
- إضافة CSS مخصص في `css/app.css` لمنع الانزياح الأفقي وتحسين عرض البطاقات.
- تحديث الإصدار إلى `1.0.90` و `versionCode 90`.
- تحديث `android-update.json`.
- تحديث `manifest.json`.

## الرسائل
- إلغاء الجدول العريض في صفحة الرسائل.
- عرض الرسائل بشكل فقاعات محادثة مناسبة للهاتف.
- إبقاء زر الإرسال ونفس دالة `sendMessage` بدون تغيير منطق Firestore.
- حماية النصوص الطويلة من دفع الصفحة خارج الشاشة.
- تمرير المحادثة تلقائيًا لآخر رسالة بعد العرض.

## الإشعارات
- إلغاء جدول الإشعارات العريض.
- عرض الإشعارات كبطاقات صغيرة.
- ترتيب أزرار تفعيل التنبيهات وتصفير العداد واختبار العداد.
- تحويل `unsupported` إلى نص عربي مفهوم: `غير مدعومة على هذا الجهاز`.
- إبقاء دوال التفعيل والتصفير والعداد الحالية كما هي.

## ما لم يتم تغييره
- لم يتم تغيير قواعد Firestore.
- لم يتم تغيير إرسال الرسائل.
- لم يتم تغيير تخزين أو قراءة الإشعارات.
- لم يتم تغيير السداد أو الفواتير أو الطلبات.
- لم يتم تغيير منطق Android Bridge.

## الفحوصات المطلوبة بعد التطبيق
افتح التطبيق ثم افحص:
- صفحة الرسائل.
- صفحة الإشعارات.
- عدم وجود سحب أفقي.
- عدم خروج الجدول أو البطاقة خارج الشاشة.
- إرسال رسالة كتاجر وعميل.
- أزرار التنبيهات والعداد.
